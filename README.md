# iris

iris is a cross platform, easy to use and customizable wallpaper manager.

<br>

## Features
- Customizable
- Easy to use
- Cross platform
- Low memory overhead and almost 0% CPU usage
- Support for remote wallpapers as well as local wallpapers
- Free & Open Source
